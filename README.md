# Company Landing Page

A simple, responsive landing page for the company, built with HTML and Tailwind CSS.

## Features

- Fully responsive design (mobile, tablet, desktop)
- SEO-optimized meta tags
- Clean, modern UI with gradient accents
- Contact form (frontend only)
- Fast loading (no external dependencies except Tailwind CDN)

## Structure

- `index.html` – main landing page
- (Optional) `assets/` – for images, fonts, etc.

## Local Development

Open `index.html` directly in a browser.

## Deployment

This page is designed to be deployed on GitHub Pages or any static hosting service.

### GitHub Pages

1. Create a new repository on GitHub.
2. Push the contents of this folder to the `main` branch.
3. Go to repository Settings → Pages.
4. Select source branch `main` (or `gh-pages`) and root folder.
5. Save; your site will be live at `https://<username>.github.io/<repo-name>/`

### Custom Domain

If you have a custom domain, add a `CNAME` file with the domain name and configure DNS accordingly.

## Content Updates

Currently using placeholder content. Replace with actual company information:

1. Update company name, mission, and contact details in `index.html`.
2. Replace placeholder icons and colors as needed.
3. Update meta tags (title, description, Open Graph) to match your brand.
